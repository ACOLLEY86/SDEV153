Best Practices Checklist

  • Linear site organization is the type of site organization used for the pacific trails website.

  • Linear seems to be the most appropriate for this site, although this could change based on future updates.

  • Page Layout
    • Appeals to target audience
    • Consistent site header
    • Consistent navigation area
    • Informative pape title that includes the site name
    • Page footer area includes copyright and contact email address
    • Good use of basic design principles, repetition, contrast, proximity, and alignment
    • Balance of text/graphics/white space on page
    • Good contrast between text and background
    • Home page downloads within 10 seconds on mobile device
    • Viewport meta tag used on each page
    • Media queries configure responsive page layout for smartphone and tablet display
  • Browser Compatibility
    • Displays on current version of Microsoft Edge
    • Displays on current version of Firefox
    • Displays on current version of Google Chrome
    • Displays on mobile devices including tablets and smartphones
  • Navigation
    • Main navigation links are clearly and consistently labeled
    • Navigation is easy to use for target audience
    • Navigation is structured in an unordered list
    • All navigation hyperlinks work
  • Three design practices that could be implemented in a better way
    • I added a hover color to the links on the pages
    • When opening webpage on widescreen monitor, the images stretch too much
    • There is no site map, but the links remain at the top for every page
  • One thing I would improve on this site is remove the border box holding in the pages.
